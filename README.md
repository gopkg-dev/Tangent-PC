### 协议功能

#### 登录相关

- ❌ 账号登录
- ✅ 扫码登录
- ✅ 缓存登录
- ✅ 修改登录状态

#### 群相关

- ❗ 群消息解析

  - [X]  普通消息
  - [X]  At人
  - [X]  普通文本
  - [X]  图片消息 (解析MD5)
  - [X]  图文消息
  - [X]  回复
  - [ ]  语音消息
  - [ ]  卡片消息
  - [ ]  红包消息
- :white_check_mark: 发送消息

  - [X]  普通消息

    - [X]  At人
    - [X]  普通文本
  - [X]  图片消息
  - [X]  图文消息
  - [X]  回复
  - [ ]  语音消息
  - [ ]  卡片消息
- ✅ 撤回消息
- :white_check_mark: 已读群消息
