### 协议功能

#### 登录相关

- ❌ 账号登录
- ✅ 扫码登录
- ✅ 缓存登录
- ✅ 修改登录状态

#### 群相关

- ❗ 群消息解析

  - [x] 普通消息
    - [x] At人
    - [x] 普通文本
  - [x] 图片消息 (解析MD5)
  - [x] 图文消息
  - [ ] 语音消息
  - [ ] 卡片消息
  - [ ] 红包消息

- ❌ 发送消息

- ✅ 撤回消息

- :white_check_mark: 已读群消息
